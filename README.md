# 📊 Sales & Revenue Analysis Dashboard

## Project Overview
This project analyzes sales and revenue data using Power BI. The dashboard provides interactive visualizations that help businesses monitor sales performance, identify top-selling products, compare regional performance, and make data-driven decisions.

## Objectives
- Analyze overall sales performance
- Monitor monthly revenue trends
- Identify top-performing products
- Compare sales across regions
- Track profit and order metrics
- Build an interactive dashboard using slicers
  
## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dataset
The project uses the **Sample Superstore Dataset**, which contains:

- Order ID
- Order Date
- Customer Name
- Product Name
- Category
- Region
- Sales
- Profit
- Quantity

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Total Quantity Sold
- Average Order Value

## Dashboard Features
- KPI Cards
- Monthly Revenue Trend
- Sales by Category
- Sales by Region
- Top 10 Products
- Profit by Category
- Customer Segment Analysis
- Interactive Slicers

## Key Business Insights
- Technology category generated the highest sales.
- West region achieved the highest revenue.
- Consumer segment contributed the largest share of sales.
- The top 10 products accounted for a significant portion of total revenue.
- Monthly sales showed strong growth during the holiday season.
